# Online Whiteboard (Flask + Fabric.js)

Single-page online whiteboard built with Python Flask, Fabric.js, and Tailwind CSS CDN.

## Features

- Freehand draw
- Rectangle, circle, line
- Eraser
- Color picker
- Brush size slider
- Undo / redo
- Clear canvas
- Download as PNG

## Run Locally

```bash
cd "/home/prayas-sawwalakhe/Desktop/Programs/My Projects/online-whiteboard"
python3 -m venv .venv
.venv/bin/python -m pip install flask
.venv/bin/python app.py
```

Open: `http://127.0.0.1:5000`
